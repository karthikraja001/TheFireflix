# Features
* Download Movies Easily
* View Movies Review (Also TV Shows)
* Watch Trailers
* Search For any Movies or TV Shows
* Add Movies to Your Favourite List
* View Trending Movies
* Simple UI
* Added Google Sign In
* Responsive UI
